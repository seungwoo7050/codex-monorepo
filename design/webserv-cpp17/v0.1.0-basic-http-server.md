# webserv-cpp17 v0.1.0 - 기본 HTTP 1.0 서버 설계

## 개요
- 단일 TCP 연결에서 HTTP/1.0 요청을 읽고 고정된 200 OK 응답을 반환한다.
- 첫 연결 처리 후 프로세스를 종료해 기본 동작을 검증한다.

## 목표 및 범위
- 포트: 기본 8080, 인자로 변경 가능.
- 요청: GET 헤더만 단순히 수신하며 바디는 무시한다.
- 응답: text/plain, "Hello from webserv v0.1.0" 문자열 반환.
- 연결: 단일 요청 처리 후 소켓 닫고 종료.

## 외부 동작
- 서버 실행 후 curl로 접속 시 200 OK와 메시지를 받는다.
- 추가 요청을 보내면 프로세스가 이미 종료되어 연결 실패가 발생한다.

## 내부 설계
- `createListenSocket`: IPv4, SO_REUSEADDR 설정 후 bind/listen.
- `respondOnce`: accept → recv → send 순서로 처리, 오류 발생 시 stderr로 한국어 메시지 출력.
- main: 포트 파싱 후 리슨 소켓 생성, 단일 응답 처리 후 종료.

## 테스트 전략
- tests/test_webserv.sh
  - 백그라운드 서버 실행 후 curl로 응답 본문 검사.
  - 추가 대기 후 프로세스 종료 여부 확인.

## 빌드/실행 참고
- CMake 3.16+ 필요, `cmake -S . -B build && cmake --build build`.
- 실행 예: `./build/webserv 9090` 후 `curl http://127.0.0.1:9090`.
