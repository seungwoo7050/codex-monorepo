# minishell-cpp17 v0.2.0 – 환경 변수 확장 및 기본 빌트인

## 목표
- `$VAR` 형태의 환경 변수 확장을 지원한다.
- `cd`, `exit`, `env` 빌트인을 제공한다.

## 범위
- 입력 한 줄에 대해 환경 변수 치환 후 공백 기반 토큰화.
- `cd`는 인자로 전달된 경로나 `HOME`으로 이동하고 현재 경로를 출력한다.
- `env`는 현재 환경 변수를 표준 출력으로 노출한다.
- `exit`는 지정된 종료 코드를 사용해 셸 프로세스를 종료한다.
- 외부 명령은 기존과 동일하게 `fork/execvp` 경로로 실행한다.

## 구현 메모
- `expandVariables`에서 `[A-Za-z_]`로 시작하고 영숫자/언더스코어로 이어지는 키만 확장한다.
- 빌트인은 `runBuiltin`에서 처리하며, `exit`의 경우 즉시 반환하여 프로세스 종료 코드를 전달한다.
- 빌트인이 아닌 경우 기존 실행 경로를 유지하고 종료 코드를 부모 프로세스 반환값으로 전달한다.

## 테스트
- `tests/env_expansion.sh`: 깨끗한 환경에서 `$FOO`가 올바르게 치환되는지 확인.
- `tests/builtin_cd_env.sh`: `cd` 출력과 `env` 출력 확인.
- `tests/builtin_exit_status.sh`: `exit 7` 호출 시 프로세스 종료 코드 전달 확인.
- 기존 `tests/run_echo.sh`를 통해 외부 명령 실행 유지 확인.

## 후속 과제
- 따옴표/이스케이프 처리와 파이프 라인 지원은 v0.3.0 이후 단계에서 다룬다.
